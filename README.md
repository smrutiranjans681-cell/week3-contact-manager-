# Week 3 Project: Contact Management System

A Python dictionary-backed Contact Management System featuring strict data validations, robust file persistence strategies with rolling backup fallbacks, functional category classification, partial matching searches, and clean CSV metrics exportation capabilities.

## Features
- Full CRUD cycle pipeline execution.
- Advanced parameter cleansing via regex formatting verification tools.
- Auto-saves runtime configuration state structures into local data stores asynchronously.

## Setup Instructions
1. Run application: `python contacts_manager.py`
2. Fire integration test matrix loops: `python test_contacts.py`
